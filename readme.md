# 不完美作战 (Imperfect Combat)

> 🎮 类幸存者波次生存 Roguelike · 三职业 · 卡牌构筑 · Boss 讨伐

纯 Canvas + JavaScript 手写的浏览器 Roguelike 游戏，适配移动端 360×640 原生分辨率，支持微信/抖音直接打开。

---

## 🚀 快速开始

```bash
# 1. 安装依赖
npm install

# 2. 启动服务器
npm start

# 3. 浏览器打开
open http://localhost:8080/game.html
```

访问 `http://localhost:8080/editor.html` 可打开配置编辑器，自由调整卡牌、Boss、波次等参数。

---

## 🎯 游戏简介

在不完美世界中，你是一名孤身对抗怪潮的战士。从三职业中选择一个，在波次中生存，通过抽卡强化自己，击败 Boss 甚至挑战无尽模式。

---

## ⚔️ 三大职业

| 职业 | 图标 | 定位 | 特性 |
|------|------|------|------|
| **修仙大仙** | 🧘 | 远程·飞剑 | 专属飞剑系统，集齐剑仙子卡觉醒追踪飞剑，7剑光解锁万剑归宗 |
| **忍术忍者** | 🥷 | 近战·暴击 | 高暴击率，快速攻击，冲刺灵活 |
| **超能力者** | 🧠 | 控场·念力 | 时间静止大招，群体冰冻控场 |

## 🃏 卡牌 & 羁绊系统

18 张卡牌，6 种羁绊类型，集齐 3 张同类型触发羁绊：

| 羁绊 | 效果 | 卡牌 |
|------|------|------|
| 😤 **狂战士** | 攻击力 +8 | 狂怒、斩杀、战吼 |
| 🗡️ **刺客** | 攻击速度 +25% | 背刺、暗影步、毒刃 |
| 🔮 **法师** | 射程 +30 | 火球、奥术飞弹、魔法盾 |
| ❄️ **冰霜** | 攻击附带冰冻减速 | 冰箭、冰霜新星、寒冰护甲 |
| 👟 **疾行** | 解锁移动速度 | 冲刺、疾风步、神行 |
| 🌀 **闪避** | 解锁闪避能力 | 闪避、幻影步、瞬移 |

商店刷新消耗 30 金币，每波结束自动弹出抽卡界面。

## 👹 Boss 系统

每 5 波随机出现 Boss，击败掉落装备：

| Boss | 波次 | 特殊掉落 |
|------|------|----------|
| 👹 山贼头目 | 波 5 | 🦺 铁甲（减伤+10%） |
| 🗡️ 暗影刺客 | 波 10 | 🗡️ 匕首（攻击+8） |
| 🛡️ 铁甲将军 | 波 15 | 🛡️ 护盾（生命+60） |
| ❄️ 冰霜女巫 | 波 20 | 💎 冰晶（冰冻+1.5s） |
| 😈 不灭魔王 | 波 25 | 🏆 最终通关 |

无尽模式（波 30+）：Boss 每 5 波动态强化，血量×1.5/伤害×1.3 递增。

## 📊 难度系统

10 档难度，在开始界面选择：

| 难度 | 说明 |
|------|------|
| 难1~4 | 简单模式，怪物削弱 |
| 难5 | 标准难度 |
| 难6~10 | 逐步变难，怪物增强 |

---

## 🎒 其他系统

- **💰 宝箱系统**：击杀怪物概率掉落，打开随机获得增益（生命/攻击/速度/范围/暴击/防御）
- **👾 精英怪**：波次中随机出现精英战士/射手/巫师，击杀掉落金币
- **🏪 装备槽**：Boss 掉落装备存入装备栏，最多携带多个道具
- **⏰ 时间静止**：超能力者专属大招，全屏冰冻
- **🤖 AI 自动战斗**：内置 AI 模式，自动走位+买卡+战斗
- **📦 编辑器**：`editor.html` 可视化编辑卡牌/Boss/波次/羁绊配置

## 🛠️ 技术架构

```
imperfect-combat/
├── server.js          # Express 静态服务器
├── package.json
└── public/
    ├── game.html      # 游戏前端页面（360×640 移动端适配）
    ├── game.js        # 主游戏逻辑（~165KB，单文件）
    ├── editor.html    # 可视化配置编辑器（PC 端）
    └── editor.js      # 编辑器逻辑
```

- **Canvas 渲染**：纯 Canvas 2D 绘制，无第三方游戏框架
- **Web Audio API**：程序化生成音效，无外部音频文件
- **A* 寻路**：怪物智能追踪
- **响应式缩放**：自动适配屏幕尺寸
- **localStorage**：宝箱数量、编辑器配置本地持久化
