# 不完美作战 (Imperfect Combat)

> 类幸存者波次生存 + Roguelike 卡牌 + 三职业系统

## 游戏简介

**不完美作战** 是一款基于 Canvas 的浏览器 Roguelike 生存游戏。选择修仙大仙、忍术忍者或超能力者，在怪物波次中生存，通过抽卡强化自己，击败 Boss 通关。

## 快速开始

```bash
npm install
npm start
```

浏览器打开 `http://localhost:8080/game.html`

## 核心系统

- **三职业**：修仙大仙（远程飞剑）、忍术忍者（近战暴击）、超能力者（念力控场）
- **卡牌系统**：6 种羁绊类型（战士/刺客/法师/冰霜/疾行/闪避），集齐触发羁绊效果
- **商店抽卡**：4 选 1 抽卡，花费金币刷新
- **Boss 系统**：每 5 波出 Boss，击败掉落装备
- **装备掉落**：Boss 掉落护甲、武器、饰品等装备
- **修仙飞剑系统**：修仙大仙专属，收集剑仙子卡觉醒飞剑追踪
- **难度系统**：10 档难度（难1~难10）
- **AI Bot**：内置 AI 自动战斗模式

## 技术栈

- 纯 JavaScript + Canvas，无框架
- Node.js + Express 静态服务
- 360×640 移动端原生分辨率
